MetaMask 연결 테스트 v3



  

    
⚠️ MetaMask가 응답하지 않습니다

    
아래 순서대로 시도하세요:

    
    
      
  1. Chrome 완전 종료 후 다시 열기 (Ctrl+Shift+Q)
    2. 
      
  2. MetaMask 아이콘 클릭 → 잠금 해제
    3. 
      
  3. 이 페이지 다시 열고 "지갑 연결" 클릭
    4. 
    

    
그래도 안 되면:

    
    
      
  1. MetaMask 아이콘 클릭 → 우측 상단 ⋮ 메뉴
    2. 
      
  2. "연결된 사이트" 또는 "Connected sites" 클릭
    3. 
      
  3. "사이트 수동 연결" 또는 "Manually connect" 클릭
    4. 
      
  4. 현재 사이트(localhost) 선택 → 연결
    5. 
      
  5. 이 페이지 새로고침
    6.
